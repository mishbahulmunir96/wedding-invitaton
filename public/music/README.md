# Background Music

Letakkan file musik latar di folder ini dengan nama: `background.mp3`

Path yang dipakai oleh MusicPlayer: `/music/background.mp3`

Rekomendasi:
- Format: MP3
- Durasi: 2-5 menit (akan di-loop otomatis)
- Volume sudah di-set 40% di kode (lihat `src/components/MusicPlayer.tsx`)

Jika ingin mengganti nama file, edit `src` di `src/components/MusicPlayer.tsx`.
