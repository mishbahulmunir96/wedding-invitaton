# Images

Folder untuk foto-foto undangan:
- `hero.jpg` — foto utama HeroSection (background atau foreground)
- `groom.jpg` — foto mempelai pria (untuk CoupleSection)
- `bride.jpg` — foto mempelai putri (untuk CoupleSection)
- `gallery-1.jpg` ... `gallery-N.jpg` — foto-foto galeri

Path akan tersedia sebagai `/images/nama-file.jpg` di komponen Next.js.

Rekomendasi:
- Format: JPG atau WebP
- Resolusi: minimal 1200x800 untuk hero, 600x800 untuk profil
- Optimasi: compress agar < 500KB per gambar
